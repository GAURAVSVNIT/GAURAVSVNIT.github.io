Abstract

Word Grouping involves identifying cohesive sequences of words into semantically meaningful units — called Local Word Groups (LWG). We model the word-grouping problem as a token classification problem and fine-tune multilingual Transformer encoder-only models for the task.

We evaluate MuRIL, XLM-Roberta, and IndicBERT v2 and report Exact Match accuracy on the test data. Our best model — MuRIL — achieves 58.18% exact match accuracy on the test set and ranks 1st among all participating teams.

Key Contributions
  • A simple and effective BIO token-classification pipeline for Indic word grouping
  • Fine-tuning and evaluation of three multilingual pretrained encoders (MuRIL, XLM-R, IndicBERT v2) with class-weighted loss to mitigate the dominant O-label bias
  • A concise error analysis highlighting model strengths and typical failure modes, with practical hyperparameters extracted from the implementation
  • Data augmentation with 5,000 sentences from a publicly available Hindi annotated dataset using rule-based LWG finder
Methodology

We treat grouping as token classification using BIO tags — B (begin group), I (inside group), O (outside). Input sentences are tokenized using the pretrained model's tokenizer, with word-level labels aligned to subword tokens using word_ids().

To address class imbalance — where most tokens are labeled O — we compute inverse-frequency class weights and apply them via a custom weighted cross-entropy loss, improving B/I token recall by 1–2% absolute.

After token-level prediction, we reconstruct grouped sentences by concatenating wordpieces within each group, joined with the __ separator matching the submission format.

Length sensitivity: shorter sentences (≤20 words) achieve 63.27% EM vs. 20.00% for long sentences (>40 words), showing clear degradation on complex inputs. Annotation inconsistencies in multiword expressions (e.g., होता है grouped 29× but ungrouped 7×) place a fundamental ceiling on achievable accuracy.

Future Directions
  • Ensemble combining MuRIL and XLM-R outputs via voting or reranking
  • Sequence-to-sequence formulations that directly produce grouped outputs, alleviating subword-label alignment issues
  • Larger models or adapters for better generalisation on long compounds
  • Synthetic data augmentation simulating real-world punctuation/whitespace noise

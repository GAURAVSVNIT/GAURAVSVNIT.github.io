## Structure

```
gauravsvnit.github.io/
├── index.html          ← Main portfolio homepage
├── README.md
├── gec/
│   ├── index.html      ← GEC project page
│   └── paper.pdf       ← GEC research paper
└── wg/
    ├── index.html      ← WG project page
    └── paper.pdf       ← WG research paper
```

## Setup

This is a static GitHub Pages site. Push to `main` branch and GitHub will serve it automatically at `https://gauravsvnit.github.io`.

To add your papers:
1. Drop your `paper.pdf` files into the `gec/` and `wg/` folders
2. Edit the `index.html` in each folder to fill in your project description, methodology, and results

## Customization

- **Bio / skills**: Edit `index.html` → the `#about` section
- **Project descriptions**: Edit `gec/index.html` and `wg/index.html`
- **Contact links**: Edit `index.html` → the `#contact` section (update email, LinkedIn URL)
- **Add more projects**: Duplicate a project folder and add a new card in `index.html`
